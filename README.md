# Using classification to predict iris species within the Iris dataset

The idea is taken from excellent blog by Bahrudin Hrnjica (https://bhrnjica.net)
Windows hosting service added with OWIN/WebApi support.

*Dependencies*
* C#/.NET 4.7.1
* CNTK 2.4.0
* WInApi 5.2.4
* Owin 4.0
* Topshelf 4.0.3
* log4net 2.0.5
